HTML 가이드

CSS 작성시 HTML의 단독 태그를 사용하지 않도록 한다. 잘못 된 예시 : h1 {color:#000}올바른 예시 : .title {color:#000} 예시와 같은 방식으로 CSS를 작성하며, CSS의 Depth는 최소화 하도록 한다.
HTML의 작성시 부모 태그를 만들어서 각 해당별 블럭 속성을 가지게 작성한다.

텍스트 기본 정의

Default : 미지정 시 기본 적용값 {font-weight:300}

Medium 미디엄 {font-weight:500}

Bold 볼드 {font-weight:700}

List Type 01

유의사항

  • 어두운 바닥에 신분증 앞면이 보이도록 놓아주세요.
  • 가이드 영역에 맞추어 촬영해주세요.

  • 빛반사에 주의하세요.

    정보 확인이 어렵거나, 훼손 또는 유효하지 않은 신분증은 거절되거나 이후 거래가 제한될 수 있습니다.
